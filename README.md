# Personal Website

My personal website built with Next.js, TypeScript, and CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Structure

```
├── app/
│   ├── blog/           # Blog posts
│   ├── projects/       # Projects
│   ├── page.tsx        # Home
│   └── layout.tsx      # Root layout
└── public/            # Assets
```

Built with Next.js and v0
